#FireBird DACView
FireBird DACView是一个完整的工控组态软件框架，包括数据的采集、处理和展示。其核心功能―部件（Component），实现了控制的聚合、封装和隔离，使得终端用户构建大型控制系统简单明了且直观。
在软件设计上，DACView采用了面向对象的设计原则，虚拟继承不仅简化了系统复杂度，而且易于维护和扩展。

#功能：
##单元（unit）
控制的最基本单元是单元（unit），每个单元实现其不同功能，如加减乘除运算、布尔值的与或非运算、PID等。单元之间的数据链接由用户生成，运行时选定。

##部件（Component）
部件。部件是一个可容纳多个部件和简单单元的容器，可无限嵌套。部件可被封装，封装后的部件其外部表现与简单单元一样，实现了控制的隔离。
